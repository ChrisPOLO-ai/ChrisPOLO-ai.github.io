# 📚 Jiyong's Workspace

지용의 개인 웹 프로젝트 모음입니다. 교육용 게임과 학급 관리 도구를 제공합니다.

## 🎯 프로젝트 구성

### 1. 메인 페이지 (index.html)
- 모던한 글래스모피즘 디자인
- 유동적인 배경 애니메이션
- 각 프로젝트로 연결되는 대시보드

### 2. AI 트레이너 게임 (deeplearning.html)
딥러닝의 기본 개념을 재미있게 배울 수 있는 교육용 게임입니다.

**주요 기능:**
- 8단계 레벨 시스템
- 학습 데이터, 라벨링, 신경망 등 딥러닝 핵심 개념 학습
- 실시간 점수 및 연속 정답 추적
- 배지 획득 시스템
- 각 문제마다 자세한 설명 제공

**학습 내용:**
- 레벨 1: 학습 데이터 만들기
- 레벨 2: 데이터 라벨링
- 레벨 3: 신경망 이해하기
- 레벨 4: 학습 과정
- 레벨 5: 예측하기
- 레벨 6: 정확도 높이기
- 레벨 7: 과적합 이해하기
- 레벨 8: 검증 데이터

### 3. 자리 배치 프로그램 (sittingarrangement.html)
학급 자리 배치를 쉽게 관리할 수 있는 스마트 도구입니다.

**주요 기능:**
- 학생 수 조정 (1~50명)
- 랜덤 이름 자동 생성
- 학생 이름 개별 수정 가능
- 드래그 앤 드롭으로 책상 위치 자유롭게 이동
- 랜덤 자리 배정 (셔플 알고리즘)
- 시각적 피드백 및 애니메이션
- 반응형 디자인

## 🚀 사용 방법

1. **시작하기**
   ```
   index.html 파일을 브라우저로 열어주세요.
   ```

2. **AI 트레이너 게임**
   - 각 레벨의 질문을 읽고 알맞은 답을 선택하세요
   - 정답을 맞히면 점수와 배지를 획득합니다
   - 8개 레벨을 모두 완료하면 딥러닝 마스터!

3. **자리 배치 프로그램**
   - 학생 수를 입력하고 "적용" 버튼 클릭
   - 왼쪽 패널에서 학생 이름 수정 가능
   - 책상을 드래그하여 원하는 위치로 이동
   - "자리 배정하기" 버튼으로 랜덤 배치
   - "책상 초기화" 버튼으로 기본 배치 복원

## 💻 기술 스택

- **HTML5** - 구조
- **CSS3** - 스타일링 (글래스모피즘, 애니메이션)
- **JavaScript (ES6+)** - 인터랙션 및 로직
- **Tailwind CSS** - 유틸리티 스타일링
- **Font Awesome** - 아이콘

## 🎨 디자인 특징

- **글래스모피즘 (Glassmorphism)**: 반투명하고 세련된 UI
- **유동적 배경**: 부드럽게 움직이는 그라데이션 blob 애니메이션
- **반응형 디자인**: 다양한 화면 크기에 최적화
- **마이크로 인터랙션**: 호버, 클릭 시 즉각적인 피드백
